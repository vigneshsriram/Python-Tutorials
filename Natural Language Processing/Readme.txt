Files included:
1) finding_dory_reivew.csv : A .csv file which contains the reviews of the movie "finding Dory" and the label of the review which can be positive/negative/neutral

2)positive-words.txt : A .txt file containg a list of positive words.

3)negative-words.txt : A .txt file containg a list of negativewords.

4) NatutalLanguagePricessing.py / NatutalLanguagePricessing.ipynb contain the main code 

Each of the review in the csv file is analysed and depending on the number of the positive/negative words in the review we find out the actual label of the review and check ones are correctly marked and which ones are wrongly marked.

The positive words are taken from the positive-words.txt and negative word from negative-words.txt
